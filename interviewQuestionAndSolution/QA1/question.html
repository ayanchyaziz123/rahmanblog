Is Unique - Algorithm Problem

Implement an algorithm to determine if a string has all unique characters. What if you cannot use additional data structures?

Example:


Input: "abcdefg"
Output: true

Input: "hello"
Output: false

Approach:

To solve this problem, you can use different approaches:

  • Using a Set or Hash Map: Iterate through the string and store each character in a set. If a character is found in the set, return false. If you reach the end of the string without finding duplicates, return true.
  • Without using additional data structures: Compare every character with every other character in the string, checking for duplicates. This approach has a time complexity of O(n2).

Pseudocode:


// With additional data structures
def isUnique(st):
    check = []
    for s in st:
        if s in check:
           return False
        check.append(s)
    return True
// Without additional data structures
function isUniqueNoDS(str) {
    for (let i = 0; i < str.length; i++) {
        for (let j = i + 1; j < str.length; j++) {
            if (str[i] === str[j]) {
                return false;
            }
        }
    }
    return true;
}
