Longest Common Prefix Explained

The Longest Common Prefix problem is a common challenge in algorithm design. It requires finding the longest common starting substring among a set of strings. This problem often appears in coding interviews and helps strengthen your understanding of string manipulation and data structures.

What is the Longest Common Prefix?

Given an array of strings, the longest common prefix is the longest substring that is common to all the strings in the array. If there is no common prefix, the function should return an empty string.

How to Approach the Problem

There are various ways to approach this problem, including:

  • Vertical scanning: Compare characters of the strings one by one until a mismatch is found.
  • Horizontal scanning: Compare each string with the current longest prefix until the longest common prefix is found.
  • Divide and conquer: Split the array of strings and find the common prefix for each half recursively.
  • Binary search: Use binary search on the length of the prefix to find the longest common prefix.
  • Trie: Construct a Trie and find the longest prefix based on the Trie structure.

Approach 1: Vertical Scanning

In the vertical scanning approach, we compare characters of each string in the array at the same index until we find a mismatch. Here's how it works:

Vertical Scanning Implementation in Python

# Python code to implement Longest Common Prefix using Vertical Scanning
class Solution:
    def longestCommonPrefix(self, strs):
        if not strs:
            return ""
        
        for i in range(len(strs[0])):
            for string in strs[1:]:
                if i >= len(string) or string[i] != strs[0][i]:
                    return strs[0][:i]
        return strs[0]

# Example usage
sol = Solution()
strs = ["flower", "flow", "flight"]
print("Longest Common Prefix (Vertical Scanning):", sol.longestCommonPrefix(strs))  # Output: "fl"

Approach 2: Using Trie

A Trie, also known as a prefix tree, can efficiently store strings and allows for quick retrieval of common prefixes. Here's how to implement the solution using a Trie:

Trie Implementation in Python

# Python code to implement Longest Common Prefix using Trie
class TrieNode:
    def __init__(self):
        self.children = {}
        self.is_end = False

class Trie:
    def __init__(self):
        self.root = TrieNode()
    
    def insert(self, word):
        node = self.root
        for char in word:
            if char not in node.children:
                node.children[char] = TrieNode()
            node = node.children[char]
        node.is_end = True

    def longest_common_prefix(self):
        prefix = ""
        node = self.root
        while node and not node.is_end and len(node.children) == 1:
            char = list(node.children.keys())[0]
            prefix += char
            node = node.children[char]
        return prefix

class Solution:
    def longestCommonPrefix(self, strs):
        trie = Trie()
        for string in strs:
            trie.insert(string)
        return trie.longest_common_prefix()

# Example usage
sol = Solution()
strs = ["flower", "flow", "flight"]
print("Longest Common Prefix (Trie):", sol.longestCommonPrefix(strs))  # Output: "fl"

Common Applications

The Longest Common Prefix problem has various applications, including:
